# World Wrapped Coin (WWC)

A cross-chain, global stable dollar router for digital assets.

## Features

- Multi-chain transaction cost mirroring
- Universal currency conversion to stable dollar
- Global API/Smart Contract interface
- Open-source for community contribution

## Getting Started

1. Clone the repo:
   ```
   git clone https://github.com/zthrigger/Wwc.git
   ```
2. Install dependencies and build contracts/scripts as needed.

## Structure

- `contracts/` - Solidity smart contracts
- `oracle/` - Oracle scripts for gas/fee data
- `sdk/` - Client libraries
- `config/` - Country to stablecoin mapping

## Example Contract Usage

See `contracts/StableDollarRouter.sol` for a minimal router contract example.
